# AI Food Assistant 🍳🤖

Интеллектуальный помощник по рецептам и кулинарии. 

## Основные возможности:
1. **Поиск рецептов**: Большая база рецептов для любых блюд.
2. **Готовка из того, что есть**: Напишите, какие продукты у вас есть в холодильнике, и ИИ предложит, что из них можно приготовить.
3. **Распознавание по фото и видео**: Загрузите фото продуктов или видео процесса, и ИИ подскажет рецепт или распознает, что перед ним.

## Выбранный стек
- **Платформа**: Веб-приложение (Frontend + Backend)
- **Фронтенд**: React (через Vite) или Next.js
- **Бэкенд / ИИ**: Node.js или Python (FastAPI) с интеграцией Gemini API для работы с текстом, фото и видео.
- **Стилизация**: Красивый современный Vanilla CSS, адаптивный дизайн для телефонов и ПК.
